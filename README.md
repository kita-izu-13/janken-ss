## 苦労や学び<br>
・　　機能が増えてくるにつれて、<br>
　　　それまでの記述ではうまくいかない部分もあったり、<br>
　　　修正したりしないといけないこともありました。<br>

・　　エラー発生時に見直して失敗しやすいパターンも<br>
　　だんだんと把握できるようになってきました。<br>
