## 苦労や学び<br>
・Java（Servletやmodel）とJSPのそれぞれの役割の違いに、<br>
　最初は戸惑いました。<br>
　それぞれで保存したものが連携しているわけではなく、<br>
　取得したりインポートしてやっと使用できるということを<br>
　自分に叩き込むようにして失敗を繰り返し慣れていきました。<br>

・実装していくにつれて、「こうだったらいいな」を形にするため、<br>
　次々と機能を増やしていきました。<br>
　その時点で学習しているものを活かせるように、<br>
　アウトプット用の作品として制作しました。<br>
　
・それぞれの関係性を考慮すると難しい部分もありましたが、<br>
　想定のものが形になるという達成感と楽しさがありました。<br>
